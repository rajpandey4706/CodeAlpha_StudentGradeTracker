# 🎓 Student Grade Tracker

A Java console-based application developed to manage student records efficiently. This project allows users to add, view, search, update, and delete student records while automatically calculating grades based on marks. It also generates a summary report showing the average, highest, and lowest marks.

## ✨ Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student
- ✏️ Update Student Marks
- 🗑️ Delete Student
- 🎯 Automatic Grade Calculation (A, B, C, D)
- 📊 Display Summary Report
- 📈 Calculate Average, Highest and Lowest Marks

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/rajpandey4706/CodeAlpha_StudentGradeTracker.git
```

2. Open the project in IntelliJ IDEA or Eclipse.

3. Compile the project.

```bash
javac StudentGradeTracker.java
```

4. Run the application.

```bash
java StudentGradeTracker
```
